bool json_talk_topic::gen_responses ( dialogue &  d) const

Callback from dialogue::gen_responses, it should add the response from here into the list of possible responses (that will be presented to the player).

It may add an arbitrary number of responses (including none at all).

Returns
true if built in response should excluded (not added). If false, built in responses will be added (behind those added here).
